## ✨ 初衷 · Why This Blog

### 📝 自用分享
> 博客是我整理知识、记录开发经验的笔记本。  
> 无论是学习笔记、bug 解决过程，还是常用代码片段，我都将其沉淀在这里，  
> 一方面方便自己查阅，另一方面也希望对你有所启发。

### 🕰️ 留作纪念
> 每一行代码，都是我技术成长的足迹。  
> 从调试的挣扎到项目的落地，这些笔记将成为未来宝贵的回忆。
